## 開発環境
### node.js

production APIに叩く場合

```
node-dev s_app_production.js
```

port:8081が埋まっている時の対処
```
lsof -i :8081
kill -9 PID
```

DEV APIに叩く場合

```
node-dev s_app_dev.js
```
### flask(API)
```
python line_analysis.py
python user.py
python add_db_instart.py
```
### docker
### マイクロサービスで立ち上がるもの(フロントを含む)
* user.py
* add_db_instart.py
* app.js（フロント）

```
$ cd API
$ docker-compose up -d
```
node.jsアクセス<br>
http://localhost:82
### マイクロサービスで立ち上がるもの(APIのみ)
* user.py
* add_db_instart.py

```
$ cd API
$ docker-compose -f docker-compose-1.yml up -d
```
